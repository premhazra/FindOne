# FindOne

**Find Skilled Professionals Instantly**

A modern React + Tailwind CSS frontend for connecting clients with verified skilled workers across India — powered by AI matching, real-time booking, and secure payments.

## Tech Stack

- **React 18** with Vite
- **Tailwind CSS** for utility-first styling
- **Framer Motion** for animations
- **React Router** for client-side routing
- **Zustand** for state management
- **Lucide React** for icons
- **Axios** for HTTP requests

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
├── app/            # Router & app-level config
├── assets/         # Static assets
├── components/     # Common & UI components
├── config/         # Site & app constants
├── data/           # Static data (categories, features, etc.)
├── features/       # Feature-based modules (landing, auth, dashboard, etc.)
├── hooks/          # Custom React hooks
├── layouts/        # Layout shells (Main, Auth, Dashboard)
├── lib/            # Route guards & utilities
├── services/       # API service layer
├── store/          # Zustand stores
└── utils/          # Helper functions
```
